# LLM Based Resume Scorer

This is a LLM Based Resume Scorer this model is used to get the ats score for user given resume with respect to the Job description provided.
Contents:
  1. Resume
  2. Job Desccription
  3. Company name

### Upload the resume in the given section
### Attach the job description of the job role.

Here the LLM will process and provide the resume score how relevant the resume is if the resume is more relavant you can go with it else it will suggest you the changes that you need to do it 

How to use the file:
1. Download the resume_scorer_st.py file
2. install the requirements
3. run the code using the command : streamlit run resume_scorer_st.py
